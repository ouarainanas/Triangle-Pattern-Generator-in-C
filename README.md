# triangle pattern generator (c)
console program that generates different types of triangles using stars

# what is this
i made this project to practice loops and nested loops in c. the program lets you pick any type of triangle and choose the size and it will generate it using stars.

# features
* 8 triangle types to choose from
* isosceles pointing up and down
* right aligned left and right
* hollow isosceles
* diamond shape
* user picks the size

# types
1. isosceles (pointing up)
2. isosceles inverted (pointing down)
3. right-aligned left
4. right-aligned left inverted
5. right-aligned right
6. right-aligned right inverted
7. isosceles hollow
8. diamond

# how to run
run the program using your c compiler or IDE (like codeblocks or vs code)

# example
triangle pattern generator 1 isosceles 2 isosceles inverted 3 right left 4 right left inverted 5 right right 6 right right inverted 7 hollow 8 diamond
choice: 1 lines: 5

# limits
* only uses stars (*)
* simple console output
